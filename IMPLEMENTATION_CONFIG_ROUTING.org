#+AUTHOR:

* Scope
Define configuration + CLI controls for market data source selection and routing.

* Config shape (minimal, capability-based)
- =equity_sources= ["eodhd", "twelve_data", "alphavantage", "marketstack"]
- =crypto_sources= ["coingecko"]
- =fx_sources= ["frankfurter"]
- =source_settings.<name>= is a TOML table, parsed by each source.
- =rate_limits.<name>= is optional and shared:
  - =max_requests=
  - =window_seconds=

* CLI
- =--prices live|cached|pinned= (already described in implementation doc)
- Optional: =--equity-sources=, =--crypto-sources=, =--fx-sources= for overrides.

* Files
- =src/config.rs= (add new config fields as generic maps)
- =src/main.rs= (wire CLI options)
- docs in =README.org= or new doc section
